# Meteocons Weather Icons

Download animated Lottie JSON files from:
https://github.com/basmilius/weather-icons

Place the "fill" style Lottie JSON files in this directory.
The app will fall back to Material Icons if these files are missing.

Required files (see MeteoconMapper.kt for full mapping):
- clear-day.json
- clear-night.json
- partly-cloudy-day.json
- partly-cloudy-night.json
- overcast.json
- fog.json
- drizzle.json
- rain.json
- extreme-rain.json
- sleet.json
- extreme-sleet.json
- snow.json
- extreme-snow.json
- thunderstorms.json
- thunderstorms-rain.json
- thunderstorms-extreme-rain.json
- partly-cloudy-day-drizzle.json
- partly-cloudy-night-drizzle.json
- partly-cloudy-day-rain.json
- partly-cloudy-night-rain.json
- partly-cloudy-day-snow.json
- partly-cloudy-night-snow.json
- not-available.json
